发表于 2018-07-18 | 分类于 LeetCode题解 | 阅读次数:

Implement strStr()

Description

Implement strStr()
Return the index of the first occurrence of needle in haystack, or -1 if needle is not part of haystack.

Example 1:

Input: haystack = “hello”, needle = “ll”
Output: 2

Example 2:

Input: haystack = “aaaaa”, needle = “bba”
Output: -1

思路

  1. Brute force
    已知str串和pattern串，在str串的每个位置都对pattern串进行遍历，一旦失配，回到pattern串起点，在str串的下一个位置重新匹配。
  2. KMP字符串匹配

    KMP算法

    KMP算法思路

    KMP算法是基于有穷自动机的改进，通过记录当前字符串的最长前缀、后缀，使在失配时可以回到之前有用的第若干个字符，不必从头开始匹配。
    如果文本串的长度为n，模式串的长度为m，那么匹配过程的时间复杂度为O(n)，算上计算next的O(m)时间，KMP的整体时间复杂度为O(m + n)。

    LPS数组

    LPS数组为longest proper prefix which is also suffix，即最长公共前缀后缀。
  • 前缀：串’aabaab’的前缀即为’a’,’aa’,’aab’,’aaba’,’aabaa’。（不带最后一个字符）
  • 后缀：串’aabaab’的后缀即为’b’,’ab’,’aab’,’baab’,’abaab’。（不带第一个字符）
  • LPS数组中储存的是字符串的LPS信息，lps[i]为前i+1个字符所组成的字符串的最长公共前缀后缀的长度。
  • lps[0] = 0
  • lps[i-1] = k的含义是：若字符串在位置i处失配，则应该直接回到第k个字符处进行匹配，即模式串向右移动的位数 = 已经匹配的字符数 - 失配字符的上一位字符的最大长度值。

    当i=4时，字符串为str[0]-str[4]，即aabaa，其前缀为’a’,’aa’,’aab’,’aaba’，其后缀为’a’,’aa’,’baa’,’abaa’，则最长公共前缀后缀为’aa’，因此lps[4]=2。
    同理，字符串’aabaab’的lps=[0,1,0,1,2,3]

    NEXT数组

    Next数组是基于LPS数组求得的，将LPS数组向右移动一位并设next[0]=-1即为next数组。
  • next[i] = k的含义是：若字符串在位置i处失配，则应回到第k个字符处进行匹配，即模式串向右移动的位数 = 失配字符的位置 - 失配字符对应的next值

    对NEXT数组的优化

    pat:   [a, a,b,a,a,b,a,a]
    next: [-1,0,1,0,1,2,3,1]
    若在第7个字符b处失配，则待匹配字符一定不是b，但根据next数组，应回到pat[3]处继续匹配，但是pat[3]也是b，一定会失配，这样就会重复迭代。因此在生成next数组的时候，不能允许pat[j]=pat[next[j]]，如果出现这种情况，要修改此处的next值，递归令next[j]=next[next[j]]（相当于在生成next数组时就在此处失配）。
